# java_service

```yaml
# Example use
dependencies:
  - role: java_service
    vars:
      java_service_name: 'awesome-java-app'
      java_service_description: 'Awesome Java App as a service'
      java_service_exec_args: '-jar /some/path/awesome-java-app.jar'
      java_service_log_file: '/some/path//awesome-java-app.log'
```
